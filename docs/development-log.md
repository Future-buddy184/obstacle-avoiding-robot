# Development Log

This document records the chronological progress of the obstacle-avoiding robot project. It serves as a timeline of major milestones, design decisions, testing, and hardware upgrades throughout development.

---

## 2026-06-19 — Project Started

### Milestones

* Created the GitHub repository.
* Defined the project goals.
* Planned the initial project architecture.
* Created the development roadmap.

### Outcome

The project was initialized with documentation before hardware assembly began.

---

## 2026-06-24 — Hardware Received

### Milestones

* Received the initial hardware components.
* Inspected all components for physical damage.
* Verified that all required components were present.

### Outcome

Hardware was ready for assembly and testing.

---

## 2026-06-25 — Robot Assembly

### Milestones

* Assembled the robot chassis.
* Installed both BO gear motors.
* Mounted the Arduino Nano.
* Mounted the ultrasonic sensor and SG90 servo.

### Outcome

Mechanical assembly completed successfully.

---

## 2026-06-25 — Initial Hardware Testing

### Milestones

* Tested the SG90 servo motor.
* Tested the HC-SR04 ultrasonic sensor.
* Verified Arduino Nano operation.
* Tested individual motor movement.

### Outcome

All individual hardware components operated successfully.

---

## 2026-06-26 — Motor Driver Investigation

### Milestones

* Encountered inconsistent motor behavior.
* Performed extensive electrical testing using a multimeter.
* Verified wiring continuity.
* Tested motors individually.
* Isolated the suspected fault to the L298N motor driver.

### Outcome

Decided to replace the L298N with a TB6612FNG motor driver for improved reliability and efficiency.

---

## 2026-06-27 — Hardware Upgrade Ordered

### Milestones

* Ordered a TB6612FNG motor driver.
* Ordered a two-cell 18650 battery holder.
* Ordered two 18650 Li-ion batteries.

### Outcome

The project entered the hardware upgrade phase.

---
