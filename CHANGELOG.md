# Version 3.0.0 (Stable)

## Added

- TB6612FNG motor driver integration
- 2×18650 Li-ion battery power system
- Smooth servo scanning
- Filtered ultrasonic distance measurements
- Adaptive turning algorithm
- Stable autonomous obstacle avoidance

## Fixed

- Resolved L298N reliability issues
- Eliminated Servo library and PWM timer conflict
- Improved turning stability
- Reduced false obstacle detection
- Improved navigation consistency

## Documentation

- Updated project architecture
- Added Version 3 documentation
- Improved repository structure
- Expanded engineering notes
- Added wiring documentation
