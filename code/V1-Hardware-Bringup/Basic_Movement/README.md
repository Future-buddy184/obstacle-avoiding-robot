# Basic Movement

## Purpose

This sketch verifies that both motors can move the robot together after individual motor testing has been completed.

It serves as the final hardware validation step before implementing autonomous obstacle avoidance.

---

## Objectives

- Verify simultaneous motor operation
- Confirm forward movement
- Validate motor synchronization
- Check overall robot wiring

---

## Hardware Used

- Arduino Nano
- TB6612FNG Motor Driver
- 2 × TT Gear Motors
- 2×18650 Li-ion Battery Pack

---

## Expected Behaviour

The robot should:

1. Move forward at a constant speed.
2. Continue moving smoothly until power is removed or the program is stopped.

Successful execution confirms that the drive system is ready for autonomous navigation.

---

## Outcome

Basic movement testing confirmed that both motors operated together correctly, marking the completion of the hardware bring-up phase and preparing the robot for obstacle-avoidance development.
